# Ultra Music Hero

Jest to gra rytmiczna oparta na znanej grze przeglądarkowej Super Crazy Guitar Mainiac Deluxe 4, posiadająca synchronizację z Youtube API i edytor poziomów.
Do zrobienia gry użyłem React, Redux i canvas
